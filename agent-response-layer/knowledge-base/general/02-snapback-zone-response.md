# 스냅백 위험구역 대응 지침

> 출처 유형: 일반 공개 안전관리 매뉴얼 (국제 계류 안전 관행 기반)
> 문서 등급: GENERAL
> Knowledge Base 태그: `scope=general`, `topic=snapback`, `lang=ko`

## 1. 스냅백 위험구역(데드존) 개념

계류삭이 파단될 때 로프는 장력 방향의 반대로 튕겨 나간다. 위험구역은 다음 요인으로 결정된다.

- 계류삭의 장력(tensionEstimateKn)과 저장 탄성에너지
- 로프 재질(합성섬유 로프는 강선보다 스냅백 반경이 크고 위험)
- 계류 기하: 볼라드와 선박 페어리드(fairlead) 사이 각도
- 로프 직경·MBL

## 2. 위험구역 내 인원 통제 절차

1. 파단 위험 판정 즉시 데드존 경계를 대시보드 지도에 표시한다(반경·폴리곤은 물리 엔진이 계산).
2. 데드존 및 로프 진행 방향 연장선상에는 절대 인원을 두지 않는다.
3. 로프 장력 작업 시 작업자는 항상 데드존 밖 측면에 위치한다.
4. 위험구역 해제는 장력 정상화·라인 교체·안전관리자 확인 후에만 선언한다.

## 3. 위험도별 통제 강도

- **CAUTION**: 데드존 예상 경계에 주의 표지, 불필요 인원 접근 제한.
- **DANGER**: 데드존 완전 통제, 전 인원 이탈, 인접 작업 중단, 하역 정지.

## 4. 합성섬유 계류삭 특이사항

합성섬유 로프(synthetic-rope)는 파단 전 육안 징후가 적고, 스냅백 시 반발 속도가 매우 빨라 위험이 크다. AE 센서의 고주파 이벤트 급증과 히트카운트 상승은 내부 소선 파단 진행을 시사하므로, 합성섬유 라인의 CAUTION은 강선보다 보수적으로(더 빨리 DANGER 준하여) 대응한다.
