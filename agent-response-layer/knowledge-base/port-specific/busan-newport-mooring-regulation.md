# 부산신항 A터미널 계류 안전 운영규정 (예시)

> 출처 유형: 서비스 이용 항만사가 개별 등록하는 사내 규정집·대처 매뉴얼 (예시 데이터)
> 문서 등급: PORT-SPECIFIC (해당 항만사 전용, 일반 지침에 우선 적용)
> 적용 사이트: `siteId=yard-a`
> Knowledge Base 태그: `scope=port-specific`, `siteId=yard-a`, `lang=ko`

> ⚠️ 본 문서는 데모용 가상 규정 예시다. 실제 운영 시 각 항만사가 자사 규정집·비상연락망·
> 조직도를 반영한 문서로 교체하여 별도 S3 prefix에 등록한다.

## 1. 조직 및 비상연락 체계

| 역할 | 호출명 | 연락 채널 |
|---|---|---|
| 관제 안전관리자 | 관제데스크 | 대시보드 + SNS SMS |
| 안벽 현장반장 | A-반장 | 무전 채널 1 |
| 항만 운영실 | 운영실 | 내선 100 |
| 비상대응(소방·구조) | 비상센터 | 119 / 내선 119 |

## 2. 볼라드·계류삭 운영 기준 (사내 강화 기준)

- 본 터미널은 항만 공통 기준(볼라드당 최대 2개 라인)보다 강화하여, **컨테이너선(300m급) 접안 시 선수·선미 스프링 라인은 볼라드당 1개로 제한**한다.
- CAUTION 판정 시 사내 기준상 **10분 이내** 현장 육안 점검 결과를 관제데스크에 회신해야 한다.
- 합성섬유 로프 CAUTION이 **연속 3회(약 3분)** 이상 유지되면 DANGER 준(準)으로 격상 관리한다.

## 3. CAUTION(주의) 단계 사내 대처 절차 (yard-a 전용)

1. 관제데스크는 A-반장에게 무전 채널 1로 해당 라인 번호와 볼라드 위치를 통보한다.
2. A-반장은 10분 이내 육안 점검(마모·소선 절단·과열) 후 관제데스크에 회신한다.
3. 크레인 작업 중이면 해당 베이의 하역 속도를 50% 감속한다.
4. 풍속 14m/s 초과 시 계류삭 추가 보강(텐션 분산) 여부를 운영실과 협의한다.

## 4. DANGER(위험) 단계 사내 대처 절차 (yard-a 전용)

1. **즉시 조치**: 관제데스크는 데드존 통제를 선언하고, 해당 선석 전 크레인 하역을 **전면 중단**한다.
2. A-반장은 데드존 내 인원 이탈 완료를 무전으로 관제데스크에 보고한다(현장 대피 경보는 자동 발령됨).
3. 운영실은 선박 대리점·도선사에 연락하여 엔진 대기 및 계류 재조정을 요청한다.
4. 인접 볼라드 라인의 연쇄 파단 가능성을 점검하고, 필요 시 인접 선석 작업도 중단한다.
5. 파단 실제 발생 시 비상센터(119)에 통보하고 사고 현장을 보존한다.
6. 관제데스크는 모든 조치 시각·수치·회신 내용을 사내 안전일지에 기록한다.

## 5. 보고 양식 (관제데스크 → 운영실)

- 발생 시각(UTC/KST), 대상 lineId, 볼라드/선박, 위험도, 핵심 AE 수치
- 취한 조치와 담당자, 데드존 통제 상태, 하역 중단 여부
- 후속 계획(라인 교체·재점검 일정)
