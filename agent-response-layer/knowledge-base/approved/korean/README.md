# 운영 승인 한국 문서

운영 RAG는 아래 한국 공식 문서만 근거로 사용한다. `approved`는 AE-Sentinel의 검색 포함
승인이며, 특정 항만의 작업허가나 법률 자문을 뜻하지 않는다.

| 문서 | 운영 용도 | 적용 한계 |
|---|---|---|
| 선박설비기준 별표 21의2 | 스냅백 구역, 장력을 받는 줄 주변의 인원 노출 최소화, 계류설비·계류삭 과부하 방지 | 총톤수 3,000톤 이상 선박에 직접 적용 |
| KOMSA 해상교통안전진단 가이드(2023) | 풍·조류·파랑, 계류삭 개수·초기장력·인장강도·배치각도와 장력해석 근거 | 평가 방법론이며 비상 명령 절차가 아님 |
| 연안선망어업 안전보건 표준매뉴얼 | 계류줄 파단 위험 접근 금지, 급박한 위험 시 작업중지·대피·보고 | 어업 분야 문서이므로 일반 안전원칙만 사용 |

항만사 승인 PDF가 등록되면 `scope=port-specific`, 해당 `siteId` 문서를 이 일반 문서보다
우선한다. 입력에 없는 실장력·로프 재질·MBL·스냅백 반경은 문서 내용으로 역산하거나
단정하지 않는다.
